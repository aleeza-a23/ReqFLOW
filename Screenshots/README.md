# 📖 About The Project

ReqFLOW is an NLP-based machine learning system designed to automatically classify software requirements into:

- **Functional Requirements (FR)** - what the system should do
- **Non-Functional Requirements (NFR)** - how well the system performs

The project applies Natural Language Processing and **Naive Bayes classification** to analyze software requirement statements and predict their category accurately with probability scores.

The project includes an interactive web interface for real-time requirement prediction with visual progress bars.
